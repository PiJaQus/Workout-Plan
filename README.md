# Workout Plan App  
**Vue 3 + TypeScript**

A modern workout planning application currently in production.  
The app allows users to create personalized training plans, track exercises, and visually highlight trained muscle groups.

---

## Features

### Landing Page  
- Clean and minimal UI  
- Gender profile selection (For Him / For Her)  
- Guided onboarding into workout creation
  
### Weekly + Daily Workout Structure  
- Create multiple training weeks 
- Add specific days inside each week (e.g., Push, Pull, Legs, Full Body)  
- Each day can hold its own list of exercises  
- Flexible structure to create long-term workout cycles

### Create Exercise  
- Add custom exercises with sets, reps, and notes  
- Select targeted muscle groups using an interactive pixel-body graphic  
- Real-time preview of highlighted muscles  
- Smooth UX with modern Vue 3 components

### Trained Muscles Overview  
- Modal view showing all activated muscle groups  
- Automatic tagging (e.g., Chest, Abs, Shoulders, Legs)  
- Helps users see body engagement during training
